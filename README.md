# NWIS-downloader
Script to pull down NWIS discharge data for a subset of monitoring locations from 2000 to present (or most recent available update while NWIS/WQP 3.0 is in beta testing).

Required:
* NWIS Info > NWIS input script.Rmd: NWIS R code
* NWIS Info > NWIS stations master with flow - edited 4-9-2025.csv: list of monitoring locations to be pulled.
